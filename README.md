## Déploiement sur GitHub Pages

### Étape 1 — Créer le dépôt GitHub
1. Aller sur [github.com](https://github.com) et créer un compte (si pas encore fait)
2. Cliquer **"New repository"**
3. Nom du dépôt : `electroadib-ma` (ou votre nom d'utilisateur : `votrenom.github.io`)
4. Choisir **Public**
5. Cliquer **"Create repository"**

### Étape 2 — Uploader les fichiers
1. Dans le dépôt créé, cliquer **"uploading an existing file"**
2. Glisser-déposer tous les fichiers de ce dossier (`index.html`, `404.html`, `README.md`)
3. Cliquer **"Commit changes"**

### Étape 3 — Activer GitHub Pages
1. Aller dans **Settings** → **Pages**
2. Source : **Deploy from a branch**
3. Branch : **main** / **root**
4. Cliquer **Save**
5. Votre site sera disponible sur : `https://votrenom.github.io/electroadib-ma`

### Domaine personnalisé (electroadib.ma)
Dans Settings → Pages → Custom domain, entrez `electroadib.ma`
Puis chez votre registrar DNS, ajoutez :
```
Type  : CNAME
Nom   : www
Valeur: votrenom.github.io
```
